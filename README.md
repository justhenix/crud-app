# Asisten Lab CRUD

A simple CRUD application built with the **TALL** stack - **Tailwind CSS, Alpine.js, Laravel, Livewire**.

## Getting Started

```bash
# Install dependencies
composer install
bun install

# Setup environment
cp .env.example .env
php artisan key:generate

# Run migrations
php artisan migrate

# Start the app
php artisan serve
```

## Stack

| Layer | Tool |
|---|---|
| Styling | Tailwind CSS |
| Frontend interactivity | Alpine.js |
| Backend | Laravel |
| UI components | Livewire |
